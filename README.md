**简介**
这是一个简单的python脚本。目前实现的功能只能将某个指定csdn博客的所有博文精简网页后，只保留标题和内容，然后转为pdf，以博文标题为名称保存在本地。

**UPDATE**
新版本支持代码高亮插件。
